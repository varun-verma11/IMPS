Below is a list of available commands in the JVG debugger. A description of
the function and usage of each command follows.

list                        :Displays the current line of code that the debugger points to.   
stp			    :Steps through the program to the next line of code and displays it.
reg      		    :Press --help reg for more info
mem 			    :Press --help mem for more info
search    		    :Press --help search for more info
pc                          :Displays the current value of the program counter.
break -r                    :To add break points onto the current line
break -a                    :To remove break points
q                           :Quits the Debugger
run                         :Run the whole program till Halt
reset                       :Reset the values in the processor for the program to run again
